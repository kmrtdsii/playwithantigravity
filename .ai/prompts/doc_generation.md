# Documentation Generation Prompt

## Goal
Generate clear, concise documentation for functions, APIs, or modules.

## Guidelines
- **Format**: JSDoc (TS/JS) or GoDoc (Go).
- **Content**:
    - Brief summary.
    - Parameters (name, type, description).
    - Return values.
    - Error conditions.
    - Example usage (if complex).
- **Tone**: Professional and objective.
